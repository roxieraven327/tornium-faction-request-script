# Tornium Faction Request Script

A userscript that allows faction members to submit requests directly from Torn instead of using Discord or Tornium’s website.

## Project Status

⚠️ This project is currently **in development**.

Some functionality depends on Tornium-side behavior that is still being discussed or adjusted. Updates will be pushed once the issue is resolved.

## What This Script Does

This script provides a way for faction members to send requests directly from the Torn interface.
It removes the need to switch to Discord or navigate to Tornium’s website to submit requests.

## Features

* Submit faction requests directly from Torn
* Faster workflow for faction members
* Simple userscript setup
* Lightweight and easy to maintain

## Requirements

To run this script you will need a userscript manager such as:

* Tampermonkey
* Violentmonkey
* Greasemonkey

## Installation

1. Install a userscript manager for your browser.
2. Download or copy the script from this repository.
3. Add the script to your userscript manager.
4. Visit Torn and use the request feature once development is complete.

## Purpose

This project was created to streamline faction request submissions and make the process easier for faction members.
